Multilevel Inheritance in Python (Person → Student → Exam)

Project Overview

This project demonstrates Multilevel Inheritance in Python using three classes:
1. Person – Base class
2. Student – Inherits from Person
3. Exam – Inherits from Student and computes total marks

Class Structure
1. Class Person
- Attributes: name, age
- Methods: display1()
2. Class Student (inherits Person)
- Adds attribute: course
3. Class Exam (inherits Student)
- Adds: marks list, total
- Methods: compute_total(), display_details()

Concepts Covered:
- Multilevel Inheritance
- Constructor Overriding
- Calling parent constructors
- Data encapsulation

Conclusion:
This program demonstrates multilevel inheritance and Python OOP concepts effectively.
